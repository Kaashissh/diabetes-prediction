# Diabetes Prediction using Decision Tree

This project implements a simple machine learning model to predict diabetes using
a Decision Tree classifier. The complete workflow is written in Python and executed
inside a Jupyter Notebook.

The dataset used in this project is the **prebuilt diabetes dataset provided by
scikit-learn** (`load_diabetes`).

## Model Used
- Decision Tree Classifier

## Dataset
- Built-in Diabetes Dataset from `sklearn.datasets`
- Loaded using `load_diabetes()`

## Libraries Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
